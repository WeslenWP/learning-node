# Iniciando

## Crie uma base de dados chama postapp
#
## Defina o usuario e senha do mysql no arquivo models/db.js de acordo

```js
// Exemplo

```

#

## Comando para criação da tabela postagens

```bash
  node syncTable.js
```

# Executando

## Agora é só executar a aplicação

```bash
  node index.js
```
